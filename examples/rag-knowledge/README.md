# RAG 전문지식 (임베딩 데이터)

이상감지 시 LLM에 주입되는 반도체 FAB 도메인 전문지식입니다.
서버 시작 시 Milvus 벡터 DB에 자동 임베딩됩니다.

## 파일 목록

| 파일 | 내용 |
|------|------|
| `equipment.md` | 설비 알람 코드(A-5xx/3xx/2xx/V-1xx), 설비 유형별 특성, 연쇄 영향 패턴 |
| `process.md` | TFT/CELL/MODULE 공정 순서, WIP 관리 기준, 에이징 LOT 판단 |
| `logistics.md` | 컨베이어 부하율 기준, AGV 가동률, 병목 판단/해소 |
| `incidents.md` | 과거 사고 사례 4건 (컨베이어 과부하, AGV 동시 장애, 포토 설비 정지, 센서 오동작) |

## 활용

이상감지 에이전트가 규칙 위반을 발견하면:
1. 관련 카테고리(equipment/logistics/wip)로 RAG 검색
2. 유사도 높은 전문지식 청크를 LLM 프롬프트에 주입
3. LLM이 전문지식 기반으로 실제 이상 여부 판단 + 조치 제안

## 커스텀

`rag/knowledge/` 디렉토리에 마크다운 파일을 추가하면 서버 재시작 시 자동 임베딩됩니다.
`##` 헤더 기준으로 청크가 분할되며, 최대 1000자 단위입니다.
