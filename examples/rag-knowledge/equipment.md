# 설비 전문지식

## 알람 코드 분류

### A-5xx: 온도 관련
- A-501: 챔버 온도 상한 초과. 히터 과열 또는 냉각수 순환 불량. 즉시 정지 필요.
  - 주요 원인: 냉각수 밸브 고장(40%), 히터 제어보드 불량(30%), 온도센서 오동작(20%)
  - 조치: 냉각수 라인 점검 → 히터 제어보드 점검 → 센서 교정
  - 영향: 해당 챔버 공정 중단, 상류 WIP 적체 시작 (보통 10~20분 내)
- A-502: 챔버 온도 하한 미달. 히터 불량 또는 전원 문제.
- A-503: 온도 불안정 (변동폭 > 2도). 제어 루프 튜닝 필요.

### A-3xx: 진공 관련
- A-301: 진공도 미달. 리크 발생 가능.
  - 주요 원인: O-ring 마모(50%), 챔버 도어 씰 불량(30%), 배관 크랙(10%)
  - 진공도 회복 시간: 정상 5분, O-ring 교체 시 30분
- A-302: 펌프 이상. 드라이펌프 또는 TMP 과부하.
- A-310: 잔류가스 농도 초과. 퍼지 불충분.

### A-2xx: 공정 관련
- A-201: 공정 시간 초과. 레시피 이상 또는 엔드포인트 미감지.
- A-210: 공정 자동 중단. 다른 알람에 의한 연쇄 정지.
- A-220: 가스 유량 이상. MFC 불량 가능.

### V-1xx: 반송설비 관련
- V-101: AGV/OHT 통신 이상. 무선 AP 또는 컨트롤러 점검.
  - 동시에 여러 대 발생 시: AP 장비 점검 우선 (단일 원인일 가능성 높음)
- V-102: AGV 경로 이탈. 바닥 마커 오염 또는 센서 불량.
- V-110: OHT 레일 센서 이상.

## 설비 유형별 특성

### CVD (Chemical Vapor Deposition)
- 정상 가동률: 85~95%
- PM 주기: 주간 챔버 클리닝, 월간 부품 교체
- 비계획정지 빈도: 월 2~3회 정상, 5회 이상이면 설비 점검 필요
- 온도 안정화 시간: 15~30분 (정지 후 재가동 시)

### 포토 (Photolithography)
- 정상 가동률: 90~97% (가장 중요한 병목 설비)
- 정지 시 영향: 전체 라인 WIP에 즉시 영향
- PR 코팅 두께 균일도: ±2% 이내 정상
- 노광 에너지 변동: ±1% 이내 정상

### 식각 (Etching)
- 정상 가동률: 88~95%
- 식각률 변동: ±3% 이내 정상
- 엔드포인트 감지 실패 시: 과식각 → 불량 발생

### 검사 (Inspection)
- 처리 속도가 상대적으로 느림 → 병목 가능
- 검사 불량률 급증 시: 상류 공정 이상 의심

## 설비 연쇄 영향 패턴

### 패턴 1: 단일 설비 정지 → WIP 적체
1. 설비 DOWN → 해당 스텝 WIP 증가 시작 (5분 내)
2. 10분 후: 상류 스텝 큐 길이 증가
3. 30분 후: 컨베이어 부하 증가
4. 1시간 후: 에이징 LOT 발생 시작

### 패턴 2: 복수 설비 동시 정지
- 같은 라인 2대 이상 정지 → critical 등급
- 같은 공정 3대 이상 정지 → 라인 정지 검토 필요
- 포토+식각 동시 정지 → 전체 라인 영향

### 패턴 3: 간헐적 정지 반복
- 같은 설비가 24시간 내 3회 이상 비계획정지 → 근본 원인 미해결
- 알람 코드가 같으면: 부품 교체 필요
- 알람 코드가 다르면: 종합 진단 필요
