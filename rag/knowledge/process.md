# 공정 흐름 전문지식

## 공정 순서와 의존성

### TFT (Thin Film Transistor) 공정
1. 세정 (TFT-01): 기판 표면 이물질 제거
2. 증착 (TFT-02): 박막 형성 (CVD/PVD)
3. 포토 (TFT-03): 패턴 형성 (포토리소그래피) ← **핵심 병목**
4. 식각 (TFT-04): 패턴 외 박막 제거
5. 검사 (TFT-05): 패턴 검사, 불량 검출

### CELL 공정
1. 배향 (CELL-01): 액정 배향막 형성
2. 합착 (CELL-02): 상하판 합착
3. 절단 (CELL-03): 패널 단위 절단
4. 검사 (CELL-04): 점등 검사

### MODULE 공정
1. 편광판 (MOD-01): 편광판 부착
2. 구동IC (MOD-02): IC 본딩
3. 조립 (MOD-03): 백라이트 조립
4. 최종검사 (MOD-04): 화질/기능 최종 검사

## WIP 관리 기준

### 정상 범위
- 각 스텝 WIP: 목표 대비 80~120% → 정상
- 120~130% → 주의 (모니터링 강화)
- 130~150% → 경고 (원인 조치 필요)
- 150% 초과 → 위험 (긴급 조치, 투입 제한 검토)

### WIP 적체 판단 기준
- 유입량 > 유출량이 30분 이상 지속 → 적체 시작
- 큐 길이 30개 초과 → 병목
- 큐 대기시간 평균 60분 초과 → 심각한 병목
- 에이징 LOT (24시간 이상 체류) → 즉시 원인 확인

### 공정 간 영향 전파
- TFT 적체 → 30분~1시간 후 CELL 영향
- CELL 적체 → 1~2시간 후 MODULE 영향
- MODULE 적체 → 출하 지연 직결

## 에이징 LOT 판단

### 정상 체류 시간
- TFT 단일 스텝: 2~6시간
- CELL 단일 스텝: 1~4시간
- MODULE 단일 스텝: 1~3시간

### 에이징 등급
- 12시간 초과: 경미 (모니터링)
- 24시간 초과: 경고 (원인 확인 필수)
- 48시간 초과: 위험 (품질 영향 가능, 재검사 필요)
- 72시간 초과: 심각 (폐기 검토)

### 에이징 원인 분류
- 설비 대기: 설비 정지/PM으로 인한 대기 (가장 흔함)
- 품질 이슈: 검사 불합격으로 HOLD
- 자재 부족: 부자재/가스 부족
- 레시피 변경: 신규 레시피 적용 대기
