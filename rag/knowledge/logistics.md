# 물류 전문지식

## 컨베이어 시스템

### 부하율 기준
- 0~60%: 정상 (여유 있음)
- 60~75%: 양호 (최적 범위)
- 75~85%: 주의 (모니터링 강화)
- 85~90%: 경고 (투입 조절 필요)
- 90% 초과: 위험 (병목 발생, 즉시 조치)
- 95% 초과: 긴급 (라인 정지 위험)

### 부하율 급등 원인
1. 하류 설비 정지 → 캐리어 정체 (가장 흔한 원인)
2. 투입량 급증 → 처리 속도 초과
3. 반송 설비(AGV/OHT) 장애 → 이동 지연
4. 컨베이어 자체 고장 → 부분 정지

### 존(Zone)별 특성
- ZONE-A: 투입구 인접, 투입량에 민감
- ZONE-B: 주요 설비 군, 설비 상태에 민감
- ZONE-C: 중간 버퍼, 상류/하류 영향 동시 수신
- ZONE-D: 출구 인접, 하류 공정 속도에 민감

## 반송 설비 (AGV/OHT)

### AGV 가동률 기준
- 정상 가동률: 85% 이상 (RUN 상태 비율)
- IDLE 비율 15% 이하: 정상
- ERROR 비율 5% 이하: 정상
- ERROR 10% 초과: 점검 필요
- ERROR 20% 초과: 시스템 문제 (컨트롤러/인프라 의심)

### 동시 다발 ERROR 패턴
- 2대 이하 ERROR: 개별 차량 문제 (센서/배터리)
- 3대 이상 동시 ERROR: 인프라 문제일 가능성 높음
  - 무선 AP 장애: 특정 구역 차량들이 동시 통신 이상
  - 컨트롤러 서버 이상: 전체 차량 명령 불가
  - 네트워크 스위치 장애: 구역별 영향

### 반송 시간 기준
- 정상 반송 시간: 30~90초 (존 간 이동)
- 120초 초과: 지연 (경로 혼잡 또는 차량 감속)
- 180초 초과: 심각 지연 (병목 또는 장애)
- 300초 초과: 정지 의심 (차량 또는 경로 점검)

## 병목 판단

### 병목 존 판단 기준
- 캐리어 대기시간 300초 초과 → 병목 후보
- 동일 존 3개 이상 캐리어 대기 → 병목 확정
- 30분 이상 지속 → 해소 불가, 조치 필요

### 병목 해소 방법 (우선순위)
1. 하류 설비 상태 확인 → 설비 문제 해결
2. 우회 경로 활성화 → 다른 라인으로 분산
3. 투입 속도 조절 → 인입량 감소
4. 버퍼 존 활용 → 임시 적재
